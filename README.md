# Tokyo Trip 2025 - Interactive Itinerary Website

An interactive website for planning and exploring a 6-day trip to Tokyo, Japan (April 15-20, 2025).

## Features

- **📅 Daily Itinerary**: Browse all 6 days with detailed activities, locations, and times
- **📊 Trip Overview**: View statistics and interactive charts showing activity distribution
- **📍 Key Locations**: Explore all destinations visited during the trip
- **⭐ Recommendations**: Discover curated suggestions for dining, attractions, shopping, and experiences

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Charts**: Chart.js for interactive data visualization
- **Hosting**: Vercel (Static Site)
- **Responsive**: Mobile-friendly design for all devices

## Getting Started

### Local Development

```bash
# Start a local server
python3 -m http.server 3000

# Open in browser
http://localhost:3000
```

### Deployment

This site is deployed on Vercel and automatically updates on every push to the repository.

## Features Included

### Itinerary Tab
- Expandable daily cards showing all activities
- Location and date information for each day
- Activity type and details

### Overview Tab
- Total activities counter
- Unique locations visited
- Average activities per day
- Interactive bar chart (activities by day)
- Doughnut chart (activity types distribution)

### Locations Tab
- List of all key destinations
- Days when each location is visited

### Recommendations Tab
- 10+ curated recommendations
- Categories: Food, Attractions, Shopping, Car-related, Entertainment
- Ratings and details for each recommendation
